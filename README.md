# Snehal Patil - Interactive Portfolio

Welcome to my personal portfolio repository! This project is a highly interactive, cinematic web experience designed to showcase my skills as a software engineer and front-end developer. 

The application goes beyond a standard static site by incorporating smooth physics-based scrolling, advanced micro-interactions, and a custom web audio engine for a fully immersive user experience.

## ✨ Key Features
- **Cinematic Audio Engine:** Custom Web Audio API implementation with ambient background tracks and tactile UI sound effects (hovers, clicks, transitions).
- **Physics-Based Scrolling:** Integrated Lenis smooth scroll for a fluid, weightless scrolling experience across all devices.
- **Advanced Animations:** Complex GSAP scroll triggers and Framer Motion layout animations for dynamic content reveals.
- **Magnetic UI Elements:** Custom React hooks that create physical pull-and-release effects on interactive elements like buttons and cursors.
- **Fully Responsive Layout:** Built with a mobile-first approach, ensuring the cinematic experience translates perfectly from massive desktop monitors down to mobile screens.

## 🛠️ Tech Stack
- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **Scrolling:** [Lenis](https://lenis.studiofreight.com/)

## 🚀 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/snehal-patil03/My-Portfolio-.git
   cd My-Portfolio-
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **View the application**
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
